# Бриф: клон Zerno → сайт для ЗАЛУ (демо #2)

> Клонуємо структуру Zerno (вона універсальна). Міняємо: контент (`content/*.json`), зображення, палітру, кілька підписів/CTA. Код, CMS і адаптив переважно лишаються.
> Фото та лого — поки AI-плейсхолдери, потім замінюються на реальні фото залу та лого власника.

---

## Палітра — ОБОВ'ЯЗКОВО змінити (під реальний зал)
Кав'ярня була тепла (коричневі/крем). Зал має читатися як зал, а не перефарбована кав'ярня.
ВАЖЛИВО: беремо реальні кольори залу — у залі жовто-чорний декор (стрічка), лого чорно-біле.
- **База: графіт / майже чорний** (#1a1a1a–#222) + світлий фон для частини секцій (за потреби).
- **Акцент: жовтий** (під жовто-чорну стрічку залу, напр. #FFD400 / #F5C518).
- Лого чорно-біле — добре лягає на темний фон (можна інвертувати в біле).
- Шрифти: можна лишити, але заголовки зробити «сильнішими» (жирніше).
- Жовтий — для кнопок/акцентів/активних станів; не заливати ним великі площі (втомлює око), краще як акцент на темному.

## Маппінг секцій (кафе → зал)

1. **Hero**
   - Фон: фото залу / тренування.
   - Заголовок: напр. «Стань сильнішим щодня» / «Твій зал у центрі міста».
   - Підзаголовок: про обладнання, тренерів, графік.
   - CTA: **«Пробне заняття безкоштовно»** (замість «Забронювати столик»).

2. **Про нас / Переваги** (4 картки)
   - Сучасне обладнання / Досвідчені тренери / Групові + персональні / Зручний графік (24/7) / Душ та роздягальні — обрати 4.

3. **Секція з вкладками: Абонементи + Тренери** (за бажанням і Напрямки)
   - Вкладка «Абонементи»: Разове, Місяць безліміт, 3 місяці, Персональні — з цінами.
   - Вкладка «Тренери»: картки тренерів (фото + ім'я + спеціалізація в описі, БЕЗ ціни).
   - (Опц.) Вкладка «Напрямки»: Тренажерний зал, Групові, Кросфіт, Бокс.
   - Технічно це ті самі картки `items` (name/desc/price/image) + категорії. Тренери = items з категорією `trainers`, `price` лишаємо порожнім.
   - ВАЖЛИВО: у картці **приховувати блок ціни, якщо `price` порожній** (щоб у тренерів не було пустого місця під ціну).
   - Заголовок секції зробити нейтральним (напр. «Абонементи та тренери»), щоб вкладки мали сенс.

4. **Галерея** — фото залу, обладнання, тренувань.

5. **Відгуки** — відгуки клієнтів залу (3 шт, рівний ряд).

6. **Тренери** — реалізовано як ВКЛАДКУ в секції вище (не окрема секція). Картки: фото + ім'я + спеціалізація, без ціни.

7. **Контакти + форма** — форма «Записатися на пробне тренування» + адреса/графік/телефон/карта + соцмережі (для залу Instagram — головне).

## Технічно (для агента)
- Скопіювати проєкт Zerno в нову папку/репо.
- Оновити `content/menu.json` (→ абонементи), `gallery.json`, `reviews.json`, `contacts.json`.
- Замінити зображення в `img/` (AI-фото залу), лого.
- Оновити палітру в `css/styles.css` (CSS-змінні).
- Поправити кілька підписів/CTA + i18n рядки (UA/EN).
- CMS config (`admin/config.yml`): за бажанням перейменувати лейбли (Меню→Абонементи), поля лишаються ті ж.

## Дисципліна обсягу (v1)
Тільки: контент + фото + палітра + підписи. НЕ додавати складних нових фіч (окрім, можливо, секції тренерів). Мета — швидко зробити переконливе демо ЗАЛУ.

## Чесність при показі власнику
Сказати прямо: фото та лого — тимчасові AI-заглушки, які замінюються на реальні фото його залу та його лого. Не видавати AI-фото за реальні.
